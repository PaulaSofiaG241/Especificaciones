Software


Servidor:

Ubuntu: es una distribución Linux basada en Debian GNU/Linux, que incluye principalmente software
libre y de código abierto, Puede utilizarse en ordenadores y servidores.

Navegador

Firefox: es el navegador web que tiene por defecto Ubuntu, este navegador es software libre y de
código abierto, además cuenta no solo con su versión para escritorio sino que también para dispositivos móviles.
Este navegador necesitas como mínimo un aproximado de 250 MB.

Bases de Datos:

phpMyAdmin: es popular y una de las herramientas de administración de bases de datos más
utilizadas para MySQL y MariaDB. Es un cliente MySQL gratuito y de código abierto popular para servicios
de alojamiento web como tambien para crear, copiar, alterar, renombrar, navegar y soltar bases de datos, tablas, etc.
Es un cliente MySQL confiable para Ubuntu.

Otros:

Office para hacer presentaciones para capacitaciones, tablas para anotar datos y documentos
para después anexar al aplicativo
  • Photoshop para toda la parte de diseño de página, así como el logo e imágenes
  • Un antivirus para mayor seguridad en los ordenadores
  • Almacenamiento en la nube para guardar documentos y tenerlos de forma fácil en cualquier dispositivo
