ESPECIFICACIONES TECNICAS HARDWARE Y SOFTWARE

Redes

En la actualidad, las redes informáticas se han convertido en un elemento esencial en la
mayoría de las empresas y organizaciones, así como también en los hogares. Estas redes permiten la
conexión y comunicación de múltiples dispositivos, facilitando el intercambio de información y recursos.

Tipos de redes informáticas

Red de Área Local (LAN)
La Red de Área Local es una red que se limita a una zona geográfica reducida, como una oficina, un
edificio o una escuela. Esta red es administrada por una sola organización y se utiliza para
compartir recursos como impresoras, archivos, entre otros. En esta red, los dispositivos están
conectados mediante
cables o redes inalámbricas. Las LAN pueden ser cableadas, es decir, que los dispositivos se conectan mediante un cable
Ethernet, o inalámbricas, que utilizan la tecnología Wi-Fi para conectar los dispositivos a la red.

Red de Área Amplia (WAN)
La Red de Área Amplia una zona geográfica mayor que una LAN, como una ciudad, un país o
incluso abarca varios países. Estas redes son utilizadas para conectar múltiples LAN y
permitir la comunicación entre ellas.
Las WAN son administradas por diferentes organizaciones y suelen utilizar diferentes
tecnologías de conexión, como líneas telefónicas, satélites, cables submarinos, entre otros.

Red de Área Metropolitana (MAN)
La Red de Área Metropolitana es una red que cubre una zona geográfica mayor que una LAN,
pero menor que una WAN. Esta red se utiliza para conectar diferentes LAN ubicadas en una
misma ciudad o zona metropolitana.
Las MAN son administradas por diferentes organizaciones y suelen utilizar tecnologías como
fibra óptica, cableado Ethernet y Wi-Fi.

Red de Área de Almacenamiento (SAN)
La Red de Área de Almacenamiento se utiliza para conectar dispositivos de almacenamiento,
como discos duros, a diferentes servidores. Esta red permite el acceso a los datos almacenados
de manera más rápida y eficiente.
Las SAN utilizan tecnologías de conexión de alta velocidad, como fibra óptica, para garantizar
un acceso rápido y seguro a los datos almacenados.


Características de las redes informáticas

Topología
La topología de una red se refiere a la forma en que están conectados los dispositivos.
Las topologías más comunes son la topología en estrella, en bus y en anillo.


Protocolos
Los protocolos son un conjunto de reglas que definen la forma en que los dispositivos
se comunican entre sí. Los protocolos más comunes son TCP/IP, HTTP, FTP y SMTP.


Ancho de banda
El ancho de banda se refiere a la cantidad de datos que pueden ser transmitidos
por segundo en una red. Un ancho de banda más alto significa que la red puede transmitir más datos en menos tiempo.


Seguridad
La seguridad es una característica importante en cualquier red informática.
Se deben implementar medidas de seguridad para proteger los datos y dispositivos de posibles
amenazas externas, como virus, hackers y malware.
