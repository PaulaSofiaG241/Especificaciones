ESPECIFICACIONES TECNICAS HARDWARE Y SOFTWARE

Licencias


Tipos de Licencias

Teniendo en cuenta las especificaciones del software debo aclarar que los servidores Linux en sí mismos no
ofrecen licencias, sino que la mayoría de las distribuciones de Linux son software libre y están disponibles
para su uso y distribución de forma gratuita bajo una licencia de código abierto, como la Licencia Pública
General de GNU (GNU GPL).
Sin embargo, hay algunas excepciones en las que se puede requerir una licencia para utilizar
ciertos programas o aplicaciones en el servidor. Algunos ejemplos son:
  • Si estás utilizando software comercial en el servidor, es posible que necesites comprar una
    licencia para utilizarlo legalmente.
  • Algunas aplicaciones de software libre pueden tener licencias restrictivas, que limitan su uso
    a ciertos propósitos o exigen que cualquier software derivado también esté disponible bajo una licencia de código abierto.

Por lo tanto, es importante revisar las licencias de los programas y aplicaciones que planeas
utilizar en tu servidor Linux, para asegurarte de que no infringes ningún derecho de autor o licencia.
Generalmente, las licencias de software se pueden encontrar en la documentación o el sitio web del desarrollador del software.
Unas de las licencias más comunes que ofrece el servidos Ubuntu de Linux son:

1. Licencia Pública General de GNU (GNU GPL): es una licencia de código abierto que permite a los
usuarios utilizar, modificar y distribuir el software siempre y cuando cualquier modificación también
esté disponible bajo la misma licencia.

2. Licencia Pública General Reducida de GNU (GNU LGPL): es una licencia de código abierto similar a la
GNU GPL, pero con algunas restricciones adicionales en cuanto a la distribución de bibliotecas de software.

3. Licencia MIT: es una licencia de código abierto muy permisiva que permite a los usuarios utilizar,
modificar y distribuir el software, incluso en aplicaciones comerciales, siempre y cuando se incluya el aviso
de derechos de autor y la licencia original.

4. Licencia Apache: es una licencia de código abierto similar a la Licencia MIT, pero con algunos términos
adicionales relacionados con las patentes y la responsabilidad legal.

5. Licencia BSD: es una licencia de código abierto muy permisiva que permite a los usuarios utilizar,
modificar y distribuir el software, incluso en aplicaciones comerciales, siempre y cuando se incluya el aviso
de derechos de autor y la licencia original.

Es importante tener en cuenta que estas son solo algunas de las licencias comunes que se utilizan en el
software que se ejecuta en servidores Ubuntu, y que cada programa o aplicación puede tener su propia licencia específica.
Por lo tanto, es importante revisar las licencias de cada programa que planeas utilizar para asegurarte
de cumplir con sus términos.
